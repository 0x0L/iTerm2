The “tool” module¶

iterm2.tool.register_web_view_tool(connection, display_name, identifier, reveal_if_already_registered, url)¶

Registers a toolbelt tool that shows a webview.

Parameters:
  • connection – A connected iterm2.connection.Connection.
  • display_name – The name of the tool. User-visible.
  • identifier – A unique ID that prevents duplicate registration.
  • reveal_if_already_registered – Bool. If True, shows the tool on a duplicate registration attempt.
  • url – The URL to show in the webview.
Returns:

iterm2.api_pb2.RegisterToolResponse on success
Raises:

iterm2.rpc.RPCException if something goes wrong
