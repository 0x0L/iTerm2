The “notifications” module¶

iterm2.notifications.unsubscribe(connection, token)¶

Unsubscribes from a notification.

Parameters:
  • connection – A connected iterm2.connection.Connection.
  • token – The result of a previous subscribe call.
iterm2.notifications.subscribe_to_new_session_notification(connection, callback)¶

Registers a callback to be run when a new session is created.

Parameters:
  • connection – A connected iterm2.connection.Connection.
  • callback – A coroutine taking two arguments: an iterm2.connection.Connection and iterm2.api_pb2.NewSessionNotification.
Returns:

A token that can be passed to unsubscribe.
